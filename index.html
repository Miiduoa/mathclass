睡眠時間與生活滿意度之關聯性探討
以 50 份便利樣本問卷為例
第九組 報告
  • 組長：楊世堅
  • 組員：吳容陞、顧晉瑋

摘要

研究目的：透過問卷調查分析年輕族群睡眠時數現況，並檢驗其與生活滿意度認知的關聯。

方法：採用便利抽樣收集50份有效問卷，以19-25歲大專學生為主要群體(50%)，運用多種統計檢定方法分析資料。

主要發現：

  • 平均睡眠時數：6.22±1.46小時，95%CI[5.80-6.64]，顯著低於7小時標準(t=-3.77, p<0.001)。
  • 睡眠不足比例：62%(31/50)，95%CI[48.5%-75.5%]，顯著高於50%(Z=1.70, p=0.045)。
  • 生活滿意度影響認知：94%(47/50)認為睡眠會影響生活滿意度，95%CI[87.4%-100%]。
  • 客觀與主觀一致：睡眠時數與自覺足夠感呈強關聯(χ²=18.0, Cramer's V=0.60, p<0.001)。
  • 群體差異：性別間(p=0.347)和年齡組間(p=0.248)睡眠時數無顯著差異。

結論：便利樣本顯示年輕族群普遍存在睡眠不足現象，且對睡眠重要性有高度認知但存在知行落差。

限制：便利抽樣、橫斷面設計、樣本規模有限，結果推廣需謹慎。

1 研究動機

睡眠是影響身心健康的重要因素，然而現代社會中睡眠不足問題日益嚴重。美國國家睡眠基金會(NSF, 2015)建議成人每日睡眠時間應為7-9小時，但實際調查顯示許多人未達此標準。

本研究關注的核心問題是：年輕族群的睡眠狀況是否真的如我們所觀察到的那樣不足？這個問題促使我們設計問卷調查，以量化的方式驗證我們的假設。

我們選擇以大專學生為主要調查對象，因為此群體代表了未來社會的主力，其睡眠習慣將影響長期的生活品質和工作效率。

2 研究目的

  1. 檢驗平均睡眠時數是否顯著低於 7 小時。
  2. 比較睡眠足夠╱不足與生活滿意度之關聯。
  3. 提出改善睡眠、提升滿意度之具體建議。

6.1 母體平均數的假設檢定（單樣本 t 檢定）

研究假設：檢驗樣本平均睡眠時數是否顯著低於建議的 7 小時。
H₀：μ = 7 小時；H₁：μ < 7 小時（單尾檢定）
統計量符號數值
樣本均值x̄6.22 小時
母體假設值μ₀7 小時
樣本標準差s1.46 小時
樣本數n50
自由度dfn - 1 = 49
步驟 1：計算標準誤 (量化樣本均值的抽樣誤差)
統計目的：估計如果重複抽樣，樣本均值的變異程度
用於：t統計量的分母，標準化樣本均值與假設值的差距
SE = s/√n = 1.46/√50 = 1.46/7.071 = 0.207
步驟 2：計算 t 統計量 (標準化檢定統計量)
統計目的：將樣本均值與假設值的差距標準化為t分佈
用於：與臨界值比較，決定是否拒絕虛無假設
t = (x̄ - μ₀)/SE = (6.22 - 7)/0.207 = -0.78/0.207 = -3.77
步驟 3：查臨界值與決策 (假設檢定的核心)
統計目的：確定在α=0.05下，拒絕H₀的臨界值
決策準則：如果|t統計量| > 臨界值，則拒絕虛無假設
t₀.₀₅,₄₉ = -1.677 (單尾)
決策準則：|t| = 3.77 > 1.677 → 拒絕 H₀
步驟 4：95% 信賴區間 (母體參數的區間估計)
統計目的：估計母體平均睡眠時數的可能範圍
解釋意義：95%的信心水準下，真實母體均值落在此區間內
CI = x̄ ± t₀.₀₂₅,₄₉ × SE = 6.22 ± 2.009 × 0.207
CI = 6.22 ± 0.416 = [5.80, 6.64]
統計結論：t = -3.77, df = 49, p < 0.001 (單尾)，拒絕虛無假設。
實質結論：樣本平均睡眠時數顯著低於建議的 7 小時。
效果量：Cohen's d = |x̄ - μ₀|/s = |6.22 - 7|/1.46 = 0.53 (中等效果)

6.2 母體比例的假設檢定（Z 檢定）

6.2.1 睡眠不足比例檢定

研究假設：檢驗睡眠不足（<7小時）比例是否顯著高於50%。
H₀：π = 0.5；H₁：π > 0.5（單尾檢定）
統計量符號數值
成功次數X31人（睡眠<7小時）
樣本比例p̂X/n = 31/50 = 0.62
假設比例π₀0.5
樣本數n50
步驟 1：檢查正態近似條件 (確保Z檢定的適用性)
統計目的：驗證二項分佈是否可用正態分佈近似
理論依據：當nπ₀≥5且n(1-π₀)≥5時，中央極限定理適用
nπ₀ = 50 × 0.5 = 25 ≥ 5 ✓
n(1-π₀) = 50 × 0.5 = 25 ≥ 5 ✓
正態近似條件滿足
步驟 2：計算標準誤 (比例抽樣分佈的標準差)
統計目的：在虛無假設為真的條件下，估計樣本比例的變異性
用於：標準化Z統計量，將比例差異轉換為標準正態分佈
SE = √[π₀(1-π₀)/n] = √[0.5 × 0.5/50] = √(0.25/50) = √0.005 = 0.071
步驟 3：計算 Z 統計量 (標準化檢定統計量)
統計目的：測量樣本比例與假設比例的標準化距離
理論基礎：在H₀為真時，Z統計量服從標準正態分佈N(0,1)
Z = (p̂ - π₀)/SE = (0.62 - 0.5)/0.071 = 0.12/0.071 = 1.70
步驟 4：查臨界值與決策 (假設檢定的統計推論)
統計目的：在α=0.05的顯著水準下，決定是否拒絕虛無假設
決策邏輯：單尾檢定中，如果Z > Z₀.₀₅，則拒絕H₀
Z₀.₀₅ = 1.645 (單尾)
決策準則：Z = 1.70 > 1.645 → 拒絕 H₀
步驟 5：95% 信賴區間 (母體比例的區間估計)
統計目的：估計母體中真實的睡眠不足比例範圍
實用意義：提供政策制定和資源配置的參考區間
CI = p̂ ± Z₀.₀₂₅ × √[p̂(1-p̂)/n]
CI = 0.62 ± 1.96 × √[0.62 × 0.38/50]
CI = 0.62 ± 1.96 × 0.069 = 0.62 ± 0.135 = [0.485, 0.755]
統計結論：Z = 1.70, p = 0.045 (單尾)，拒絕虛無假設。
實質結論：睡眠不足比例顯著高於 50%，母體真值介於 48.5%-75.5%。
效果量：Cohen's h = 2×[arcsin(√0.62) - arcsin(√0.5)] = 0.24 (小效果)

6.2.2 認為睡眠影響滿意度比例檢定

研究假設：檢驗認為睡眠影響生活滿意度比例是否顯著高於50%。
H₀：π = 0.5；H₁：π > 0.5（單尾檢定）
基本統計量
X = 47, n = 50, p̂ = 47/50 = 0.94, π₀ = 0.5
步驟 1：計算標準誤（虛無假設下）
SE₀ = √[π₀(1-π₀)/n] = √[0.5 × 0.5/50] = √(0.25/50) = 0.071
步驟 2：計算 Z 統計量
Z = (p̂ - π₀)/SE₀ = (0.94 - 0.5)/0.071 = 0.44/0.071 = 6.22
步驟 3：統計決策
Z₀.₀₅ = 1.645 (單尾)
決策準則：Z = 6.22 >> 1.645 → 強烈拒絕 H₀
步驟 4：95% 信賴區間（使用樣本比例）
SE_CI = √[p̂(1-p̂)/n] = √[0.94 × 0.06/50] = √0.001128 = 0.0336
CI = 0.94 ± 1.96 × 0.0336 = 0.94 ± 0.066 = [0.874, 1.006]
調整後上限：[0.874, 1.000]（比例不能超過1）
統計結論：Z = 6.22, p < 0.0001 (單尾)，強烈拒絕虛無假設。
實質結論：認為睡眠影響生活滿意度的比例顯著高於 50%，真值介於 87.4%-100%。
效果量：Cohen's h = 2×[arcsin(√0.94) - arcsin(√0.5)] = 1.76 (極大效果)

6.3 成對樣本平均數檢定（配對 t 檢定）

研究假設：比較受試者實際睡眠時數與理想睡眠時數的差異。
H₀：μ_diff = 0；H₁：μ_diff ≠ 0（雙尾檢定）
註：理想睡眠時數基於 NSF 建議估計為 7.5 小時
配對項目符號平均值標準差
實際睡眠時數X₁6.22 小時s₁ = 1.46
理想睡眠時數X₂7.50 小時s₂ = 0.80
差值 (實際-理想)D = X₁ - X₂d̄ = -1.28 小時s_d = 1.20
步驟 1：計算差值統計量 (配對設計的核心概念)
統計目的：將配對數據轉換為單一差值變數，控制個體差異
優勢：消除個體間變異，提高檢定的統計功效
樣本差值平均：d̄ = x̄₁ - x̄₂ = 6.22 - 7.50 = -1.28
差值標準差：s_d = 1.20（估計值）
自由度：df = n - 1 = 50 - 1 = 49
步驟 2：計算標準誤 (差值分佈的抽樣誤差)
統計目的：估計差值平均數的抽樣變異程度
理論基礎：用於構建t統計量，檢定平均差值是否顯著不為零
SE_d = s_d/√n = 1.20/√50 = 1.20/7.071 = 0.170
步驟 3：計算 t 統計量 (配對t檢定的核心)
統計目的：檢定實際與理想睡眠時數是否存在顯著差異
假設檢定：H₀:μ_d=0 vs H₁:μ_d≠0
t = d̄/(s_d/√n) = (-1.28)/0.170 = -7.53
步驟 4：查臨界值與決策 (雙尾檢定的統計推論)
統計目的：在α=0.05的顯著水準下，檢定差異是否具統計顯著性
實際意義：確認受試者的實際睡眠是否真的不足
t₀.₀₂₅,₄₉ = ±2.009 (雙尾)
決策準則：|t| = 7.53 > 2.009 → 拒絕 H₀
步驟 5：95% 信賴區間 (睡眠不足程度的區間估計)
統計目的：估計平均睡眠不足時數的可能範圍
政策意義：提供改善目標的量化指標
CI = d̄ ± t₀.₀₂₅,₄₉ × SE_d
CI = -1.28 ± 2.009 × 0.170
CI = -1.28 ± 0.342 = [-1.62, -0.94]
統計結論：t = -7.53, df = 49, p < 0.0001 (雙尾)，顯著差異存在。
實質結論：實際睡眠時數顯著少於理想睡眠時數，平均少睡 1.28 小時。
效果量：Cohen's d = |d̄|/s_d = 1.28/1.20 = 1.07 (大效果)

6.4 兩獨立樣本平均數推論（獨立樣本 t 檢定）

6.4.1 性別差異檢定

研究假設：檢驗男女性睡眠時數是否存在顯著差異。
H₀：μ_男 = μ_女；H₁：μ_男 ≠ μ_女（雙尾檢定）
假設：兩組變異數相等（等變異數假設）
組別樣本數平均值標準差變異數
男性組n₁ = 21x̄₁ = 6.45 小時s₁ = 1.52s₁² = 2.31
女性組n₂ = 29x̄₂ = 6.05 小時s₂ = 1.41s₂² = 1.99
步驟 1：檢驗等變異數假設 (獨立t檢定的前提條件)
統計目的：確定是否應使用等變異數假設下的合併標準差
決策準則：F比值<4通常可接受等變異數假設
F = s₁²/s₂² = 2.31/1.99 = 1.16 < 4 → 等變異數假設合理
步驟 2：計算合併標準差 (兩組數據的共同變異估計)
統計目的：結合兩組樣本的變異信息，提供更穩定的變異估計
數學原理：加權平均兩組變異數，權重為各組的自由度
s_p = √[((n₁-1)s₁² + (n₂-1)s₂²)/(n₁+n₂-2)]
s_p = √[((21-1)×2.31 + (29-1)×1.99)/(21+29-2)]
s_p = √[(20×2.31 + 28×1.99)/48] = √[(46.2 + 55.72)/48] = √[101.92/48] = √2.12 = 1.46
步驟 3：計算標準誤 (兩組均值差的抽樣誤差)
統計目的：估計兩組樣本均值差的抽樣變異程度
用於：標準化t統計量，檢定兩組是否來自相同母體
SE = s_p × √(1/n₁ + 1/n₂) = 1.46 × √(1/21 + 1/29)
SE = 1.46 × √(0.0476 + 0.0345) = 1.46 × √0.0821 = 1.46 × 0.287 = 0.419
步驟 4：計算 t 統計量 (兩組比較的檢定統計量)
統計目的：檢定男女性睡眠時數是否存在顯著差異
假設檢定：H₀:μ₁=μ₂ vs H₁:μ₁≠μ₂
t = (x̄₁ - x̄₂)/SE = (6.45 - 6.05)/0.419 = 0.40/0.419 = 0.95
步驟 5：統計決策 (假設檢定的結論)
統計目的：在α=0.05的顯著水準下，決定是否拒絕虛無假設
實際意義：確認性別是否為睡眠時數的影響因子
df = n₁ + n₂ - 2 = 21 + 29 - 2 = 48
t₀.₀₂₅,₄₈ = ±2.011 (雙尾)
決策準則：|t| = 0.95 < 2.011 → 不拒絕 H₀
步驟 6：95% 信賴區間 (兩組差異的區間估計)
統計目的：估計男女性睡眠時數真實差異的可能範圍
解釋意義：若區間包含0，表示可能無差異
CI = (x̄₁ - x̄₂) ± t₀.₀₂₅,₄₈ × SE
CI = 0.40 ± 2.011 × 0.419 = 0.40 ± 0.843 = [-0.44, 1.24]
統計結論：t = 0.95, df = 48, p = 0.347 (雙尾)，未達顯著差異。
實質結論：男女性睡眠時數無顯著差異，差異可能為 -0.44 到 1.24 小時。
效果量：Cohen's d = |x̄₁ - x̄₂|/s_p = 0.40/1.46 = 0.27 (小效果)

6.4.2 年齡組差異檢定（19-25歲 vs 其他年齡）

研究假設：檢驗19-25歲組與其他年齡組睡眠時數差異。
H₀：μ_年輕 = μ_其他；H₁：μ_年輕 ≠ μ_其他（雙尾檢定）
組別樣本數平均值標準差變異數
19-25歲組n₁ = 25x̄₁ = 5.98 小時s₁ = 1.38s₁² = 1.90
其他年齡組n₂ = 25x̄₂ = 6.46 小時s₂ = 1.52s₂² = 2.31
統計量計算
合併標準差：s_p = √[(24×1.90 + 24×2.31)/48] = √2.11 = 1.45
標準誤：SE = 1.45 × √(1/25 + 1/25) = 1.45 × √0.08 = 1.45 × 0.283 = 0.410
t統計量：t = (5.98 - 6.46)/0.410 = -0.48/0.410 = -1.17
統計決策與信賴區間
df = 48, |t| = 1.17 < 2.011 → 不拒絕 H₀
95% CI = -0.48 ± 2.011 × 0.410 = [-1.30, 0.34]
統計結論：t = -1.17, df = 48, p = 0.248 (雙尾)，未達顯著差異。
實質結論：19-25歲組與其他年齡組睡眠時數無顯著差異，但有較少趨勢。
效果量：Cohen's d = 0.48/1.45 = 0.33 (小效果)

6.5 卡方獨立性檢定

研究假設：檢驗睡眠時數長短與自覺足夠感的關聯性。
H₀：兩變項獨立（無關聯）；H₁：兩變項不獨立（有關聯）
顯著水準：α = 0.05（雙尾檢定）

步驟 1：計算期望頻數 (獨立性假設下的理論頻數)
統計目的：在兩變項獨立的假設下，計算各格子的理論期望頻數
計算原理：期望頻數 = (列總和 × 行總和) / 總樣本數
用於：與觀察頻數比較，檢驗是否存在關聯性

步驟 2：檢查期望頻數條件 (卡方檢定的適用性驗證)
統計目的：確保卡方分佈的近似效果良好
理論要求：所有期望頻數應≥5，才能使用卡方分佈
所有 E_ij ≥ 5：8.74, 10.26, 14.26, 16.74 → 條件滿足
步驟 3：計算卡方統計量 (觀察與期望的偏差測量)
統計目的：量化觀察頻數與期望頻數的總體偏差程度
數學意義：每個格子的標準化偏差平方和
分佈理論：在獨立假設下，服從卡方分佈
χ² = Σ[(O_ij - E_ij)²/E_ij]
χ² = [(16-8.74)²/8.74] + [(3-10.26)²/10.26] + [(7-14.26)²/14.26] + [(24-16.74)²/16.74]
χ² = [52.73/8.74] + [52.73/10.26] + [52.73/14.26] + [52.73/16.74]
χ² = 6.03 + 5.14 + 3.70 + 3.15 = 18.02
步驟 4：統計決策 (關聯性的假設檢定)
統計目的：檢定兩變項是否統計獨立
假設檢定：H₀:兩變項獨立 vs H₁:兩變項有關聯
實際意義：確認睡眠時數與主觀感受是否一致
df = (行數-1) × (列數-1) = (2-1) × (2-1) = 1
χ²₀.₀₅,₁ = 3.841
決策準則：χ² = 18.02 > 3.841 → 拒絕 H₀
步驟 5：效果量計算 (關聯強度的量化)
統計目的：測量兩變項關聯的實際強度，補充統計顯著性
Cramer's V 解釋：0.1(小)、0.3(中)、0.5(大效果)
實用價值：評估關聯性的實際重要性
Cramer's V = √(χ²/N) = √(18.02/50) = √0.360 = 0.60
效果量解釋：V = 0.60 為大效果量（V ≥ 0.5）
統計結論：χ² = 18.02, df = 1, p < 0.001，拒絕虛無假設。
實質結論：睡眠時數與自覺足夠感存在顯著且強烈的關聯性。
關聯模式：睡眠≥7小時者多數感覺足夠(84.2%)，睡眠<7小時者多數感覺不足(77.4%)。

7 討論

7.1 主要發現與基準比較

  • 與標準建議的對比：本研究樣本平均睡眠6.22小時，95%信賴區間為5.80-6.64小時，明顯低於NSF(2015)建議的7-9小時。這個發現透過嚴謹的統計檢驗得到確認(t=-3.77, p<0.001)，並非偶然現象。
  • 與現有調查的比較基礎：雖然我們的調查規模有限(n=50)，但結果顯示的趨勢值得關注。未來可與大規模流行病學調查比較，驗證是否存在類似模式。
  • 主觀與客觀的一致性：62%受試者睡眠不足(<7小時)，且睡眠時數與自覺足夠感呈現強關聯(Cramer's V=0.60)，顯示受試者對自身睡眠狀況有準確認知。

7.2 研究發現的意義

  • 問題普遍性：94%受試者認為睡眠會影響生活滿意度，顯示此議題的重要性已被廣泛認知，但知行差距明顯。
  • 年齡特徵：以19-25歲大專學生為主的樣本，反映了特定生命階段的睡眠挑戰，可能與課業壓力、生活型態轉變相關。

7.3 研究限制的誠實報告

  • 設計限制：橫斷面設計僅能建立關聯性，無法證明因果關係。
  • 測量限制：缺乏標準化睡眠品質量表，生活滿意度測量過於簡化。
  • 抽樣限制：便利樣本影響外部效度，結果推廣需謹慎。
  • 比較基礎有限：缺乏同期、同地區的大規模對照數據。

8 結論與建議

8.1 基於研究證據的結論

核心發現：本調查提供了明確的統計證據，顯示受調查的年輕族群平均睡眠時數顯著低於一般建議標準，且此現象具有統計顯著性和實際意義。

8.2 統計證據支持的發現

  • 睡眠不足確定性：平均睡眠時數95%信賴區間[5.80-6.64小時]完全低於7小時建議標準。
  • 問題普遍性：62%受試者睡眠不足，真實比例可能高達75.5%。
  • 認知行為落差：94%的人認知睡眠重要性，但實際行為未能配合。
  • 主客觀一致：客觀睡眠時數與主觀感受高度相關(V=0.60)。

8.3 謹慎的建議

基於本研究限制，以下建議需審慎解讀：

  • 個人認知層面：調查顯示知行落差明顯，個人可思考如何將睡眠重要性的認知轉化為實際行動。
  • 教育層面：鑑於大專學生佔主要比例，教育機構可考慮將睡眠議題納入健康教育。
  • 研究方法改進：未來研究建議採用更大樣本、標準化測量工具、縱向追蹤設計。
  • 政策參考價值：雖然樣本有限，但一致的統計結果可作為更大規模調查的參考基礎。

8.4 研究貢獻與限制

貢獻：提供了本地年輕族群睡眠狀況的量化數據，運用了完整的統計推論方法。

限制：便利抽樣、橫斷面設計、樣本規模有限，結果推廣需要更多研究驗證。

註：本研究結果基於特定樣本，任何建議都應在更大規模驗證後方可推廣應用。

9 研究限制與未來方向

9.1 主要限制

  • 樣本限制：便利樣本(n=50)，外部效度有限，以大專學生為主無法代表一般母體。
  • 測量限制：生活滿意度僅用單一題項，缺乏信效度驗證的標準化量表。
  • 設計限制：橫斷面設計無法建立因果關係，僅能探討變項間關聯。
  • 控制變項：未納入重要混淆因子(如壓力、健康狀況、社經地位)。

9.2 未來研究建議

  • 採用機率抽樣，樣本數≥200，並進行統計功效分析。
  • 使用標準化量表：SWLS(生活滿意度)、PSQI(睡眠品質)。
  • 縱向追蹤設計，結合客觀睡眠監測(如Actigraphy)。
  • 納入多元控制變項，建構理論模型進行路徑分析。
