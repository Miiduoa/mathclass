睡眠時間與生活滿意度之關聯性探討
以 50 份便利樣本問卷為例
第九組 報告
  • 組長：楊世堅
  • 組員：吳容陞、顧晉瑋

摘要

研究目的：分析年輕族群睡眠時間現況，並探討對睡眠與生活滿意度關聯性的認知。

方法：便利抽樣收集50份問卷，以19-25歲大專學生為主(50%)，測量睡眠時數、主觀足夠感及滿意度影響認知。

有效統計發現：

  • 平均睡眠時數：6.22±1.46小時，95%CI[5.80-6.64]，顯著低於7小時(t=-3.77, p<0.001)。
  • 睡眠不足比例：62%(31/50)，95%CI[48.5%-75.5%]，顯著高於50%(Z=1.70, p=0.045)。
  • 滿意度影響認知：94%(47/50)認為睡眠會影響生活滿意度，95%CI[87.4%-100%]。
  • 主客觀一致性：睡眠時數與自覺足夠感呈強關聯(χ²=18.0, Cramer's V=0.60, p<0.001)。

方法學限制：

  • ❌ 部分分析無效：配對t檢定使用虛構數據，獨立t檢定缺乏分組數據支持。
  • ⚠️ 未進行多重比較校正，實際第一類錯誤率約30%。
  • ⚠️ 便利樣本(n=50)功效不足，僅能檢測大效果量。
  • ⚠️ 未測量真實生活滿意度，僅調查認知。

可靠結論：年輕族群確實存在睡眠不足現象，對睡眠重要性有高度認知，主客觀評估一致。

學術價值：主要為統計方法學的學習案例，展示了研究設計缺陷對分析可靠性的影響。

1 研究動機

睡眠是影響身心健康的重要因素，然而現代社會中睡眠不足問題日益嚴重。美國國家睡眠基金會(NSF, 2015)建議成人每日睡眠時間應為7-9小時，但實際調查顯示許多人未達此標準。

本研究關注的核心問題是：年輕族群的睡眠狀況是否真的如我們所觀察到的那樣不足？這個問題促使我們設計問卷調查，以量化的方式驗證我們的假設。

我們選擇以大專學生為主要調查對象，因為此群體代表了未來社會的主力，其睡眠習慣將影響長期的生活品質和工作效率。

2 研究目的

核心目標：探討年輕族群睡眠時間現況，以及對睡眠與生活滿意度關聯性的認知。

  1. 描述睡眠現況：分析樣本的睡眠時數分布，檢驗是否低於建議標準。
  2. 探討主客觀一致性：比較客觀睡眠時數與主觀睡眠足夠感的關聯程度。
  3. 分析滿意度認知：調查受試者對「睡眠影響生活滿意度」的認知普遍性。
  4. 檢驗群體差異：比較不同性別和年齡組在睡眠模式上的差異。

註：本研究重點在於睡眠時間現況分析和對睡眠重要性的認知調查，而非直接測量生活滿意度與睡眠的因果關係。

6.1 母體平均數的假設檢定（單樣本 t 檢定）

研究假設：檢驗樣本平均睡眠時數是否顯著低於建議的 7 小時。
H₀：μ = 7 小時；H₁：μ < 7 小時（單尾檢定）
統計量符號數值
樣本均值x̄6.22 小時
母體假設值μ₀7 小時
樣本標準差s1.46 小時
樣本數n50
自由度dfn - 1 = 49
步驟 1：計算標準誤 (量化樣本均值的抽樣誤差)
統計目的：估計如果重複抽樣，樣本均值的變異程度
用於：t統計量的分母，標準化樣本均值與假設值的差距
SE = s/√n = 1.46/√50 = 1.46/7.071 = 0.207
步驟 2：計算 t 統計量 (標準化檢定統計量)
統計目的：將樣本均值與假設值的差距標準化為t分佈
用於：與臨界值比較，決定是否拒絕虛無假設
t = (x̄ - μ₀)/SE = (6.22 - 7)/0.207 = -0.78/0.207 = -3.77
步驟 3：查臨界值與決策 (假設檢定的核心)
統計目的：確定在α=0.05下，拒絕H₀的臨界值
決策準則：如果|t統計量| > 臨界值，則拒絕虛無假設
t₀.₀₅,₄₉ = -1.677 (單尾)
決策準則：|t| = 3.77 > 1.677 → 拒絕 H₀
步驟 4：95% 信賴區間 (母體參數的區間估計)
統計目的：估計母體平均睡眠時數的可能範圍
解釋意義：95%的信心水準下，真實母體均值落在此區間內
CI = x̄ ± t₀.₀₂₅,₄₉ × SE = 6.22 ± 2.009 × 0.207
CI = 6.22 ± 0.416 = [5.80, 6.64]
統計結論：t = -3.77, df = 49, p < 0.001 (單尾)，拒絕虛無假設。
實質結論：樣本平均睡眠時數顯著低於建議的 7 小時。
效果量：Cohen's d = |x̄ - μ₀|/s = |6.22 - 7|/1.46 = 0.53 (中等效果)

6.2 母體比例的假設檢定（Z 檢定）

6.2.1 睡眠不足比例檢定

研究假設：檢驗睡眠不足（<7小時）比例是否顯著高於50%。
H₀：π = 0.5；H₁：π > 0.5（單尾檢定）
統計量符號數值
成功次數X31人（睡眠<7小時）
樣本比例p̂X/n = 31/50 = 0.62
假設比例π₀0.5
樣本數n50
步驟 1：檢查正態近似條件 (確保Z檢定的適用性)
統計目的：驗證二項分佈是否可用正態分佈近似
理論依據：當nπ₀≥5且n(1-π₀)≥5時，中央極限定理適用
nπ₀ = 50 × 0.5 = 25 ≥ 5 ✓
n(1-π₀) = 50 × 0.5 = 25 ≥ 5 ✓
正態近似條件滿足
步驟 2：計算標準誤 (比例抽樣分佈的標準差)
統計目的：在虛無假設為真的條件下，估計樣本比例的變異性
用於：標準化Z統計量，將比例差異轉換為標準正態分佈
SE = √[π₀(1-π₀)/n] = √[0.5 × 0.5/50] = √(0.25/50) = √0.005 = 0.071
步驟 3：計算 Z 統計量 (標準化檢定統計量)
統計目的：測量樣本比例與假設比例的標準化距離
理論基礎：在H₀為真時，Z統計量服從標準正態分佈N(0,1)
Z = (p̂ - π₀)/SE = (0.62 - 0.5)/0.071 = 0.12/0.071 = 1.70
步驟 4：查臨界值與決策 (假設檢定的統計推論)
統計目的：在α=0.05的顯著水準下，決定是否拒絕虛無假設
決策邏輯：單尾檢定中，如果Z > Z₀.₀₅，則拒絕H₀
Z₀.₀₅ = 1.645 (單尾)
決策準則：Z = 1.70 > 1.645 → 拒絕 H₀
步驟 5：95% 信賴區間 (母體比例的區間估計)
統計目的：估計母體中真實的睡眠不足比例範圍
實用意義：提供政策制定和資源配置的參考區間
CI = p̂ ± Z₀.₀₂₅ × √[p̂(1-p̂)/n]
CI = 0.62 ± 1.96 × √[0.62 × 0.38/50]
CI = 0.62 ± 1.96 × 0.069 = 0.62 ± 0.135 = [0.485, 0.755]
統計結論：Z = 1.70, p = 0.045 (單尾)，拒絕虛無假設。
實質結論：睡眠不足比例顯著高於 50%，母體真值介於 48.5%-75.5%。
效果量：Cohen's h = 2×[arcsin(√0.62) - arcsin(√0.5)] = 0.24 (小效果)

6.2.2 認為睡眠影響滿意度比例檢定

研究假設：檢驗認為睡眠影響生活滿意度比例是否顯著高於50%。
H₀：π = 0.5；H₁：π > 0.5（單尾檢定）
基本統計量
X = 47, n = 50, p̂ = 47/50 = 0.94, π₀ = 0.5
步驟 1：計算標準誤（虛無假設下）
SE₀ = √[π₀(1-π₀)/n] = √[0.5 × 0.5/50] = √(0.25/50) = 0.071
步驟 2：計算 Z 統計量
Z = (p̂ - π₀)/SE₀ = (0.94 - 0.5)/0.071 = 0.44/0.071 = 6.22
步驟 3：統計決策
Z₀.₀₅ = 1.645 (單尾)
決策準則：Z = 6.22 >> 1.645 → 強烈拒絕 H₀
步驟 4：95% 信賴區間（使用樣本比例）
SE_CI = √[p̂(1-p̂)/n] = √[0.94 × 0.06/50] = √0.001128 = 0.0336
CI = 0.94 ± 1.96 × 0.0336 = 0.94 ± 0.066 = [0.874, 1.006]
調整後上限：[0.874, 1.000]（比例不能超過1）
統計結論：Z = 6.22, p < 0.0001 (單尾)，強烈拒絕虛無假設。
實質結論：認為睡眠影響生活滿意度的比例顯著高於 50%，真值介於 87.4%-100%。
效果量：Cohen's h = 2×[arcsin(√0.94) - arcsin(√0.5)] = 1.76 (極大效果)

6.4 研究設計限制的統計影響

多重比較問題：本研究進行了7項假設檢定，在α=0.05下，犯第一類錯誤的機率實際上為：
P(至少一個錯誤) = 1 - (0.95)⁷ = 0.30
建議：應使用Bonferroni修正，調整α = 0.05/7 = 0.007
統計功效不足：n=50的樣本對於檢測中等效果量(d=0.5)，統計功效約為0.55，低於建議的0.80。
影響：可能錯過真正存在的效果（第二類錯誤風險高）

6.5 卡方獨立性檢定

研究假設：檢驗睡眠時數長短與自覺足夠感的關聯性。
H₀：兩變項獨立（無關聯）；H₁：兩變項不獨立（有關聯）
顯著水準：α = 0.05（雙尾檢定）
觀察頻數表 (Observed Frequencies)
自覺足夠自覺不足列總和
≥7 hrO₁₁ = 16O₁₂ = 3R₁ = 19
<7 hrO₂₁ = 7O₂₂ = 24R₂ = 31
行總和C₁ = 23C₂ = 27N = 50
步驟 1：計算期望頻數 (獨立性假設下的理論頻數)
統計目的：在兩變項獨立的假設下，計算各格子的理論期望頻數
計算原理：期望頻數 = (列總和 × 行總和) / 總樣本數
用於：與觀察頻數比較，檢驗是否存在關聯性
E₁₁ = (R₁ × C₁)/N = (19 × 23)/50 = 8.74
E₁₂ = (R₁ × C₂)/N = (19 × 27)/50 = 10.26
E₂₁ = (R₂ × C₁)/N = (31 × 23)/50 = 14.26
E₂₂ = (R₂ × C₂)/N = (31 × 27)/50 = 16.74
步驟 2：檢查期望頻數條件 (卡方檢定的適用性驗證)
統計目的：確保卡方分佈的近似效果良好
理論要求：所有期望頻數應≥5，才能使用卡方分佈
所有 E_ij ≥ 5：8.74, 10.26, 14.26, 16.74 → 條件滿足
步驟 3：計算卡方統計量 (觀察與期望的偏差測量)
統計目的：量化觀察頻數與期望頻數的總體偏差程度
數學意義：每個格子的標準化偏差平方和
分佈理論：在獨立假設下，服從卡方分佈
χ² = Σ[(O_ij - E_ij)²/E_ij]
χ² = [(16-8.74)²/8.74] + [(3-10.26)²/10.26] + [(7-14.26)²/14.26] + [(24-16.74)²/16.74]
χ² = [52.73/8.74] + [52.73/10.26] + [52.73/14.26] + [52.73/16.74]
χ² = 6.03 + 5.14 + 3.70 + 3.15 = 18.02
步驟 4：統計決策 (關聯性的假設檢定)
統計目的：檢定兩變項是否統計獨立
假設檢定：H₀:兩變項獨立 vs H₁:兩變項有關聯
實際意義：確認睡眠時數與主觀感受是否一致
df = (行數-1) × (列數-1) = (2-1) × (2-1) = 1
χ²₀.₀₅,₁ = 3.841
決策準則：χ² = 18.02 > 3.841 → 拒絕 H₀
步驟 5：效果量計算 (關聯強度的量化)
統計目的：測量兩變項關聯的實際強度，補充統計顯著性
Cramer's V 解釋：0.1(小)、0.3(中)、0.5(大效果)
實用價值：評估關聯性的實際重要性
Cramer's V = √(χ²/N) = √(18.02/50) = √0.360 = 0.60
效果量解釋：V = 0.60 為大效果量（V ≥ 0.5）
統計結論：χ² = 18.02, df = 1, p < 0.001，拒絕虛無假設。
實質結論：睡眠時數與自覺足夠感存在顯著且強烈的關聯性。
關聯模式：睡眠≥7小時者多數感覺足夠(84.2%)，睡眠<7小時者多數感覺不足(77.4%)。

• ✅ 可信結果：單樣本檢定、比例檢定、關聯分析（基於實際收集數據）
• ⚠️ 估計結果：性別差異（缺乏分組詳細數據）
• ❌ 無效分析：配對t檢定（虛構理想睡眠數據）

重要結論：僅前四項檢定提供有效的統計證據

7 討論

7.1 主要發現與基準比較

  • 與標準建議的對比：本研究樣本平均睡眠6.22小時，95%信賴區間為5.80-6.64小時，明顯低於NSF(2015)建議的7-9小時。這個發現透過嚴謹的統計檢驗得到確認(t=-3.77, p<0.001)，並非偶然現象。
  • 與現有調查的比較基礎：雖然我們的調查規模有限(n=50)，但結果顯示的趨勢值得關注。未來可與大規模流行病學調查比較，驗證是否存在類似模式。
  • 主觀與客觀的一致性：62%受試者睡眠不足(<7小時)，且睡眠時數與自覺足夠感呈現強關聯(Cramer's V=0.60)，顯示受試者對自身睡眠狀況有準確認知。

7.2 研究發現的意義

  • 問題普遍性：94%受試者認為睡眠會影響生活滿意度，顯示此議題的重要性已被廣泛認知，但知行差距明顯。
  • 年齡特徵：以19-25歲大專學生為主的樣本，反映了特定生命階段的睡眠挑戰，可能與課業壓力、生活型態轉變相關。

7.3 研究限制的誠實報告

  • 設計限制：橫斷面設計僅能建立關聯性，無法證明因果關係。
  • 測量限制：缺乏標準化睡眠品質量表，生活滿意度測量過於簡化。
  • 抽樣限制：便利樣本影響外部效度，結果推廣需謹慎。
  • 比較基礎有限：缺乏同期、同地區的大規模對照數據。

8 結論與建議

8.1 研究實際貢獻的重新定位

重要澄清：本研究實際上是「年輕族群睡眠現況調查」和「睡眠重要性認知調查」，而非標題所示的真正關聯性研究。我們測量的是對關聯性的認知，而非實際關聯。

8.2 基於實際研究內容的發現

  • 睡眠現況確認：年輕族群平均睡眠6.22小時，95%CI[5.80-6.64]，確實低於7小時標準。
  • 主客觀一致性：睡眠時數與主觀感受高度一致(V=0.60)，顯示受試者有準確的自我認知。
  • 認知普遍性：94%的人認為睡眠會影響生活滿意度，顯示此認知非常普遍。
  • 知行落差：儘管認知到重要性，但實際睡眠行為未能配合。

8.3 謹慎的建議

基於現有證據，以下建議需審慎解讀：

  • 個人反思：調查顯示知行落差明顯，個人可思考如何將認知轉化為行動。
  • 教育價值：可作為統計方法學的完整應用案例。
  • 研究啟發：為未來真正的關聯性研究提供初步參考。

8.4 學術誠實與限制

研究定位：本研究的學術價值主要在於統計分析方法的完整應用，而非實質的科學發現。

重要限制：未測量真正的生活滿意度，因此無法支持標題中的「關聯性探討」。

未來方向：需要使用SWLS等標準化量表，才能進行真正的變項關聯分析。

註：學術誠實要求我們承認研究設計與標題間的落差，這也是科學研究的重要學習過程。

9 研究限制與方法學反思

9.1 統計方法學的重大問題

❌ 嚴重方法學錯誤：
• 配對t檢定使用了虛構的"理想睡眠"數據，這在統計學上完全不當
• 獨立t檢定缺乏實際的分組數據支持，僅基於估計
• 未進行多重比較校正，實際第一類錯誤率約30%

9.2 數據收集限制

  • 測量工具問題：自編問卷未經信效度檢驗，生活滿意度僅測量認知而非實際量表分數。
  • 樣本代表性：便利樣本(n=50)嚴重不足，外部效度極其有限。
  • 數據完整性：缺乏詳細的分組交叉數據，無法進行可靠的組間比較。
  • 設計缺陷：橫斷面設計無法建立因果關係。

9.3 統計功效與檢定力

功效分析：n=50檢測中等效果量(d=0.5)的統計功效僅約55%，遠低於建議的80%。
影響：可能錯過真正存在的效果，結論可靠性受限。

9.4 可靠結論與不可靠結論

結論類型可靠性原因
平均睡眠低於7小時✅ 可靠基於實際數據，效果量中等
睡眠不足比例>50%✅ 較可靠基於實際數據，但效果量小
認知影響比例極高✅ 可靠極大效果量，結果一致
主客觀睡眠高度相關✅ 可靠大效果量，卡方檢定適當
性別/年齡組差異⚠️ 不確定缺乏實際分組數據
實際vs理想睡眠差距❌ 不可靠理想睡眠數據為虛構

9.5 未來研究的必要改進

  • 樣本設計：使用機率抽樣，樣本數≥200，進行統計功效分析。
  • 測量工具：採用標準化量表：SWLS(生活滿意度)、PSQI(睡眠品質)。
  • 數據收集：完整的人口學變項交叉表，支持組間比較分析。
  • 統計方法：多重比較校正、正態性檢驗、效果量報告。
  • 研究設計：縱向追蹤，結合客觀睡眠監測技術。

9.6 學術誠實與教育價值

方法學學習：本研究的主要價值在於暴露了統計分析中的常見錯誤，提供了重要的方法學反思機會。

誠實評估：承認統計分析的局限性是科學研究的基本要求，也是統計學教育的重要組成部分。
